R package cognitoR 1.0.1 has been released

Tuesday, Apr 28, 2020 by Aleksander Dietrichson, PhD R, R-bloggers, AWS

Today we are proud to announce that the cognitoR package has been released on CRAN. The package provides password protection for your Shiny apps using the popular Cognito from Amazon Web Services. Cognito allows for simple and secure sign-up, sign-in and access control. The cognitoR package brings the power of this service into your Shiny Apps. You can let users register to access your app, or, use the AWS infrastructure to implement fine tuned access control.
